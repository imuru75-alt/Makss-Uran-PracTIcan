📁 Что входит в проект:
index.html — основной файл с обновленной логикой на React и Vanilla JS.

404.html — кастомная страница ошибки в стиле сайта.

robots.txt — файл инструкций для поисковиков.

sitemap.xml — карта сайта для индексации.

.htaccess — конфигурация сервера (редиректы, кэширование, защита).

🚀 Как запустить проект:
Открой папку с проектом в VS Code.

Нажми правой кнопкой на index.html и выбери "Open with Live Server".

Проект запустится по адресу http://127.0.0.1:5500/. Это важно, так как пути в SEO-файлах настроены именно под этот локальный адрес.

💡 Что обязательно указать в отчете (для защиты практики):
React Context API: Я реализовал глобальное состояние для уведомлений. Теперь при отправке любой формы вызывается тост-уведомление через провайдер.

Управление данными (Storage): * LocalStorage: Используется для сохранения корзины, списка «Недавно просмотренные» и автосохранения черновиков в формах (данные не пропадают при обновлении страницы).

SessionStorage: Реализован счетчик просмотров товаров за текущую сессию (сбрасывается при закрытии вкладки). Это наглядно показывает разницу между типами хранилищ.

React Portals: Использованы для рендеринга корзины, чтобы избежать конфликтов со стилями шапки сайта (Z-index и фильтры размытия).

Техническая оптимизация: Подготовлены файлы для SEO-продвижения и серверные правила .htaccess.

⚠️ Что нужно знать (Нюансы):
.htaccess: Этот файл работает только на реальных серверах (Apache). В Live Server он не будет выполнять редирект на HTTPS, но для отчета и сдачи практики он обязателен, так как показывает твои знания серверной настройки.

404 Страница: Чтобы увидеть её в действии без сервера, просто открой файл 404.html вручную.
